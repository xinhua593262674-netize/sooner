# 方案 C 最终图片包

## 在线 Demo

部署到 GitHub Pages 后，打开仓库 Pages 地址即可从 `index.html` 进入完整交互链路。

核心链路：

```text
入口场景
→ 复习专区
→ 闪卡
→ 错题
→ 变式题
→ 复习报告
```

## 页面清单

1. `01_入口_会员态.png`
   - 智能复习入口，会员态。
2. `02_入口_未开通会员态.png`
   - 智能复习入口，未开通会员态。
3. `03_推荐页_篇章考点层级.png`
   - 智能复习推荐页，支持科目切换，结构为篇 → 章 → 考点。
4. `04_背闪卡.png`
   - 统一练习播放器：闪卡阶段。
5. `05_错题再练_原题乱序.png`
   - 统一练习播放器：原题选项乱序重做阶段。
6. `06_变式题.png`
   - 统一练习播放器：变式题阶段。
7. `07_报告_建议重练.png`
   - 考点报告：建议重新练习。
8. `08_报告_已掌握.png`
   - 考点报告：已掌握。

## 核心路径

```text
入口
→ 推荐页（科目切换 / 推荐篇 / 推荐章 / 考点）
→ 背闪卡
→ 错题再练
→ 变式题
→ 考点报告
```

## 入口文案

```text
智能复习
依据做题记录，剖析错因，推送考点复习
定位错因｜巩固考点｜错题重做｜变式提升
```
